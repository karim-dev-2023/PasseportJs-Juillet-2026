# React Passport OAuth App

Ce projet est une application d’authentification OAuth réalisée avec **React**, **Node.js**, **Express** et **Passport.js**.

L’objectif est de permettre à un utilisateur de se connecter via plusieurs fournisseurs OAuth :

- Google
- GitHub

Le projet est séparé en deux parties :

```txt
react_passeport/
├── client/      # Frontend React
└── backend/     # Backend Node.js / Express / Passport.js
